	Программа project_0.ipynb запрашивает начало и конец диапазона угадывания чисел, 
затем запрашивает загаданное число и угадывает его за не более, чем за 
*количество попыток = логарифм по основанию 2 из размера диапазона угадывания* попыток.

	Описание принципа работы алгоритма:	
	Алгоритм каждый раз делит диапазон угадывание пополам, после этого сравнивает
предпалогаемое число с загаданым, если предполагаемое число меньше загаданного - 
алгоритм отбрасывает ту часть диапазона угадывания, которая меньше загаданного числа,
в противном случае - отбрасывает часть, которая больше, после чего выводит предполагаемое
число и номер попытки и цикл повторяется до тех пор,пока предполагаемое число не станет
равно угаданному, затем выводит угаданное число и количество попыток.

	Программа draft.ipynb необходима для тестирования алгоритмов угадывания. Если
подставить функцию угадывания в функцию score_game(), программа выполнит алгоритм угадывания
заданное количество раз, подставляя псевдо-случайный ряд чисел из заданного диапазона и 
выдаст среднее, максимальное, минимальное количество попыток угадывания.

	Для программы project_0.ipynb среднее количество попыток угадывания числа из диапазона
от 1 до 100 составляет 5, максимальное 7, минимальное 1
